What is Coding Club?

Coding club is a place where anyone of any experience can get with friends and peers to learn code and share projects. Each month we will focus on one coding language to learn the basics of and take on a coding project.

Girls Who Code

Girls Who Code is a nonprofit organization that provides free programs for those who identify as female or non-binary that are interested in learning the basics of coding. The organization offers a free club program for school with all online curriculum, webinars, and projects. At the moment, the Girls Who Code sub-club of coding club here at CHS is still in the process of being scheduled; information will be available soon.
