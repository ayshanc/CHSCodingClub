What is Coding Club?

Coding club is a place where anyone of any experience can get with friends and peers to learn code and share projects. Each month we will focus on one coding language to learn the basics of and take on a coding project.

Coding Club Flyer

Girls Who Code

Girls Who Code is a nonprofit organization that provides free programs for those who identify as female or non-binary that are interested in learning the basics of coding. The organization offers a free club program for school where we will learn to code from fun online tutorials and activities that will help build animations, games, and apps using Scratch, Python, Swift, HTML, CSS, and JavaScript.
