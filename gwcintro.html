What did we do on Girls Who Code HQ?

Once you sign up for a Girls Who Code account and join our club using the registration code (on the Home Page or in the Presentation), you will have access to all our tutorials, activities, and curriculum.

Our GWC Project for the time being is a Python Chatbot (can be found under "Activities and Tutorials" -> "Chatbot in Python Tutorial"). If you would like, run through the "Get Started" and sign up for a Trinket account so save your progress as we go.
